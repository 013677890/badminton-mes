# 接口需求分析

接口需求用于描述 MES 与 ERP、设备和其他系统之间的数据交互边界。本文仅做业务需求分析，不展开具体接口协议和技术实现。

## 1. ERP 接口：生产任务单读取接口

### 功能概述

生产任务单读取接口用于将 ERP 中的生产任务或生产计划读取到 MES，作为生产工单或生产任务单的来源。

### 业务流程

1. MES 按时间或人工触发读取 ERP 生产任务数据。
2. 系统校验产品、单位、数量、交期和来源单号。
3. 读取成功后生成 MES 生产工单或待确认任务。
4. 异常数据进入接口日志，由人员处理。

### 参与角色

- PMC 计划员：确认读取结果。
- 系统管理员：查看接口日志和异常。
- ERP 系统：提供生产任务数据。

### 输入输出

- 输入：ERP 任务单号、产品编码、计划数量、交期、车间、来源订单。
- 输出：MES 生产工单、读取日志、异常清单。

### 主要数据

- ERP 任务读取记录：记录来源单号和同步状态。
- 接口异常日志：记录缺失字段、重复数据和校验失败原因。

### 规则与异常

- ERP 来源单号不能重复生成 MES 工单。
- 产品编码不存在时应进入异常清单。
- 数量小于等于零的数据不允许生成生产任务。

### 接口需求

- 支持按时间范围或单号读取 ERP 任务。
- 支持记录成功、失败和重复读取状态。

### 验收标准

- 能从 ERP 读取生产任务并生成 MES 单据。
- 重复数据不会重复生成。
- 接口异常可查询和重新处理。

## 2. ERP 接口：工艺数据读取

### 功能概述

工艺数据读取接口用于从 ERP 或工艺系统读取产品工艺路线、工序和 BOM 相关数据，供 MES 工艺管理和生产派工使用。

### 业务流程

1. MES 读取 ERP 工艺数据。
2. 系统校验产品、工序、路线版本和生效状态。
3. 数据进入 MES 待确认区。
4. 工艺工程师确认后，形成 MES 工艺路线。
5. 异常数据记录接口日志。

### 参与角色

- 工艺工程师：确认读取的工艺数据。
- 系统管理员：处理接口异常。
- ERP 系统：提供工艺数据。

### 输入输出

- 输入：产品编码、工艺路线编码、工序编码、工序顺序、版本、生效日期。
- 输出：MES 工艺路线、工序信息、读取日志、异常清单。

### 主要数据

- 工艺读取记录：记录来源系统、来源编码和同步状态。
- 工艺待确认数据：记录未审核的外部工艺信息。
- 工艺接口异常：记录缺失、重复和顺序错误。

### 规则与异常

- 工序顺序必须完整且不能重复。
- 产品不存在时不能直接生成工艺路线。
- 外部工艺变更应形成 MES 新版本，不能覆盖历史版本。

### 接口需求

- 支持读取产品工艺路线和工序明细。
- 支持工艺工程师确认后生效。

### 验收标准

- 能读取外部工艺数据。
- 能形成可审核的 MES 工艺路线。
- 异常工艺数据可查询和处理。

## 3. 标准 API：计量单位写入

### 功能概述

计量单位写入接口用于允许其他系统将计量单位写入 MES，保持产品、物料和报工单位一致。

### 业务流程

1. 外部系统提交计量单位数据。
2. MES 校验单位编码、名称和状态。
3. 新单位写入基础资料，已有单位按规则更新。
4. 写入结果返回外部系统并记录日志。

### 参与角色

- 系统管理员：维护和检查基础资料。
- 外部系统：提交单位数据。

### 输入输出

- 输入：单位编码、单位名称、精度、启用状态。
- 输出：写入结果、错误信息、接口日志。

### 主要数据

- 计量单位表：记录单位编码、名称和状态。
- API 写入日志：记录请求来源和处理结果。

### 规则与异常

- 单位编码必须唯一。
- 已被业务单据使用的单位不能随意修改精度。
- 缺失编码或名称时写入失败。

### 接口需求

- 支持新增和更新计量单位。
- 支持返回成功、失败和失败原因。

### 验收标准

- 外部系统能写入单位数据。
- 异常数据不会污染基础资料。
- 写入日志可查询。

## 4. 标准 API：工单写入

### 功能概述

工单写入接口用于允许其他系统将车间生产工单写入 MES，作为生产订单模块的数据来源。

### 业务流程

1. 外部系统提交工单数据。
2. MES 校验产品、数量、交期、BOM 和工艺路线。
3. 校验通过后生成生产工单。
4. 校验失败时返回错误原因并记录接口日志。

### 参与角色

- PMC 计划员：确认外部写入工单。
- 系统管理员：处理接口异常。
- 外部系统：提交工单。

### 输入输出

- 输入：外部工单号、产品编码、计划数量、交期、客户、优先级。
- 输出：MES 工单号、写入状态、错误信息。

### 主要数据

- 外部工单写入记录：记录来源工单和 MES 工单关系。
- 工单接口日志：记录请求和处理结果。

### 规则与异常

- 外部工单号不能重复写入。
- 产品、数量和交期为必填。
- 缺少必要主数据时，工单进入异常或拒绝写入。

### 接口需求

- 支持外部系统新增生产工单。
- 支持查询写入结果。

### 验收标准

- 能通过 API 写入生产工单。
- 重复工单不会重复生成。
- 写入异常可查询。

## 5. 标准 API：生产任务单写入

### 功能概述

生产任务单写入接口用于允许其他系统将订单或指令单直接写入 MES 生产任务单。

### 业务流程

1. 外部系统提交生产任务单。
2. MES 校验产品、产线、班次、数量和时间。
3. 校验通过后生成待审核生产任务单。
4. 车间主管审核后下发现场。

### 参与角色

- 车间主管：审核外部写入任务。
- 系统管理员：处理接口异常。
- 外部系统：提交任务数据。

### 输入输出

- 输入：外部任务号、产品、产线、班次、计划数量、计划开始时间、计划结束时间。
- 输出：MES 任务单号、写入状态、异常信息。

### 主要数据

- 任务写入记录：记录外部任务和 MES 任务关系。
- 任务接口日志：记录写入状态和异常原因。

### 规则与异常

- 外部任务号必须唯一。
- 产线不存在或停用时，不允许写入有效任务。
- 任务数量不能超过来源工单允许数量。

### 接口需求

- 支持新增生产任务单。
- 支持返回任务单写入结果。

### 验收标准

- 能写入生产任务单并进入待审核状态。
- 异常数据能被拒绝或进入异常清单。
- 写入记录可追溯。

## 6. 标准 API：设备计数报工写入

### 功能概述

设备计数报工写入接口用于接收设备对接系统提交的设备计数数据，并将其用于设备计数记录或生产报工。

### 业务流程

1. 设备系统提交设备计数数据。
2. MES 校验设备、采集时间、计数值和关联任务。
3. 匹配成功后生成设备计数记录。
4. 按配置生成自动报工或待确认报工。
5. 异常数据进入设备计数异常池。

### 参与角色

- 设备管理员：维护设备接入配置。
- 产线班组长：确认设备计数报工。
- 操作工：核对自动计数结果。

### 输入输出

- 输入：设备编码、采集时间、计数值、运行状态、任务标识、工序标识。
- 输出：计数记录、报工记录、异常信息。

### 主要数据

- 设备计数接口记录：记录设备上报数据。
- 自动报工记录：记录由设备计数生成的报工。
- 设备计数异常表：记录无法匹配或重复数据。

### 规则与异常

- 重复计数数据不能重复报工。
- 未绑定当前任务的设备数据不能自动报工。
- 计数值倒退或异常跳变应进入异常池。

### 接口需求

- 支持设备系统写入计数数据。
- 支持返回处理结果和异常原因。

### 验收标准

- 能接收设备计数并形成记录。
- 能按配置生成报工或待确认报工。
- 异常计数数据可查询和处理。

## 7. 标准 API：生产完工单读取

### 功能概述

生产完工单读取接口用于允许 ERP、WMS 或其他系统读取 MES 中已审核的生产完工单数据。

### 业务流程

1. 外部系统按时间或单号请求读取完工单。
2. MES 返回已审核或已确认的生产完工单。
3. 外部系统处理入库、结案或财务核算。
4. MES 记录读取日志和读取状态。

### 参与角色

- 仓库人员：根据完工结果办理入库。
- PMC 计划员：查看工单完成情况。
- 外部系统：读取完工单数据。

### 输入输出

- 输入：时间范围、完工单号、工单号、读取状态。
- 输出：完工单号、工单号、产品、批次、完工数量、良品数量、不良数量、审核状态。

### 主要数据

- 生产完工单：记录完工结果。
- 完工读取日志：记录外部系统读取时间和结果。

### 规则与异常

- 未审核完工单不应对外提供为正式数据。
- 已作废完工单应明确标识。
- 重复读取应返回相同业务数据，不应改变完工单内容。

### 接口需求

- 支持按时间范围和单号读取完工单。
- 支持返回读取状态和业务状态。

### 验收标准

- 外部系统能读取已审核完工单。
- 读取日志可查询。
- 作废或未审核数据能被正确区分。
