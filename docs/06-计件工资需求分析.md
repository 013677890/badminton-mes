# 计件工资需求分析

计件工资模块用于根据工序单价和员工报工数量计算员工计件金额，为车间绩效和工资核算提供基础数据。

## 1. 工序计件

### 功能概述

工序计件按照不同工序的计件单价和操作工完成的合格数量计算计件金额。羽毛球生产可针对植毛、注胶、修整、检验、包装等工序设置不同计件标准。

### 业务流程

1. 管理人员维护工序计件单价。
2. 操作工在现场完成工序报工。
3. 系统按员工、工序、任务、日期汇总合格数量和不良扣减数量。
4. 系统根据计件规则计算计件金额。
5. 班组长或车间主管审核计件结果。
6. 审核后的计件数据用于绩效分析或工资核算。

### 参与角色

- 车间主管：审核计件结果。
- 产线班组长：确认员工实际作业情况。
- 操作工：查看本人计件数量和金额。
- 生产经理：查看人员绩效统计。

### 输入输出

- 输入：工序、员工、报工数量、合格数量、不良数量、计件单价、扣减规则。
- 输出：员工计件明细、计件汇总、工资核算基础数据、绩效统计。

### 主要数据

- 工序计件单价表：记录工序、产品、单价和生效日期。
- 员工报工汇总表：记录员工在各工序的有效产量。
- 计件工资明细表：记录员工、工序、数量、单价和金额。
- 计件审核记录：记录审核人、审核时间和审核意见。

### 规则与异常

- 计件数量应以审核后的有效报工数量为准。
- 不良品是否计件应按工序计件规则确定。
- 同一员工同一工序同一时间段不能重复计算。
- 计件单价变更应从生效日期起影响新数据，不影响已审核数据。
- 计件结果调整需要记录原因。

### 接口需求

- 从生产报工获取人员、工序和数量数据。
- 从工艺管理获取工序信息。
- 向报表分析提供人员绩效和工序金额统计。

### 验收标准

- 能维护工序计件单价。
- 能按员工、日期、工序计算计件金额。
- 能查看计件明细和汇总。
- 计件结果审核后不能随意修改。
