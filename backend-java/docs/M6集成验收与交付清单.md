## 最小闭环演示顺序

1. 登录并确认 Bearer 会话及车间/产线数据权限。
2. 创建或选择生产任务，生成条码并完成扫码入序。
3. 执行开工、暂停、恢复、完工，提交正常报工和反向冲销。
4. 打开报表确认净额、审计发生额、不良来源和批次追溯结果。
5. 从不良报工创建返修单，完成指派、返修记录、复检、放行或继续返修。
6. 通过小程序聚合接口读取实时看板、生产分析和产品追溯。
7. 通过看板 HTTP 接口获取快照，并使用 STOMP 订阅增量通知；断线后先重新获取完整快照再恢复订阅。
